Welcome to PDMUnit's help page

Thank you for choosing Blue Byte Systems Inc for your SOLIDWORKS PDM API development needs!

This help webpage contains extensive documentation about PDMUnit. PDMUnit is a hybrid unit and integration testing library for SOLIDWORKS PDM Professional. You can use PDMUnit to create blazingly integration tests for your add-ins and task add-ins without loading them into the vault.

Getting Started

The easiest way to get started is to use our Visual Studio PDMUnit project template. This project template is freely available on the Visual Studio marketplace.

If you prefer to create your own test project, we highly suggest you use a .NET 4.7.2 console application.

Install PDMUnit from nuget.org using this command

Copy
Install-Package PDMUnit

You can use the code sample below to start:
